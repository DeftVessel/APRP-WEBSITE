HOW TO USE

▸ Use the sidebar to navigate sections and edit content.
▸ Every save button writes directly to Supabase — changes show on the archive instantly and persist after refresh.
▸ Use RAW JSON to bulk-paste or export the full dataset.
▸ Portrait and map images: get a direct URL from imgur.com, paste into the image URL field per-president / per-election.
▸ Save buttons write to the Supabase database. Wait for the green save confirmation before navigating away.
