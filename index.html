HOW TO USE

▸ Use the sidebar to navigate sections and edit content.
▸ Every save button writes directly to this page's memory — changes show on the archive instantly (no reload needed).
▸ Use RAW JSON to bulk-paste or export the full dataset.

▸ Changes auto-save to JSONBin — wait for ✓ SYNCED in the header before navigating away.
