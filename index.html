HOW TO USE

▸ Use the sidebar to navigate sections and edit content.
▸ Every save button writes directly to this page's memory — changes show on the archive instantly (no reload needed).
▸ Use RAW JSON to bulk-paste or export the full dataset.
▸ Portrait and map images: get a direct URL from imgur.com, paste into the image URL field per-president / per-election.
▸ Changes auto-save to JSONBin — wait for ✓ SYNCED in the header before navigating away.

• Cols A/B/C = House district, name, party (LEFT side)
• Cols D/E/F = Senate/House district, name, party (RIGHT side)
• Bold rows (CLASS 1/2/3 SENATOR) = Senate seats
• Commonwealth headers (COLUMBIA, PHOENIX etc) = grouping
Rows with VACANT or empty name are kept as VACANT.
